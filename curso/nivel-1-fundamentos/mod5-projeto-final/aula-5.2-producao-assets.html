Produção de Assets com IA

Crie todos os elementos visuais e sonoros do seu projeto.

🎬 Workflow de Produção

Etapa 1: Gerar Imagens-Chave

Use DALL-E 3, Midjourney ou Leonardo para cada cena do roteiro:

Exemplo de prompt para Cena 1 (Hook):

"Professional product shot of [PRODUTO], modern minimalist style, soft studio lighting, floating in space with subtle gradient background, 4K, cinematic, product photography"

Dicas:

  • • Gere 3-4 variações de cada cena
  • • Mantenha consistência de estilo (mesmo prompt base)
  • • Salve em resolução máxima (1024x1024 ou maior)
  • • Organize em pastas por cena

Etapa 2: Animar Imagens (Image-to-Video)

Use Runway ou Pika Labs para dar movimento às imagens:

Cena 1 - Produto em Destaque:

Motion prompt: "Camera slowly rotating around product, subtle zoom in, professional reveal"

Cena 2 - Problema:

Motion prompt: "Hands typing on keyboard, slight camera movement, ambient office setting"

Cena 3 - Demonstração:

Motion prompt: "Interface elements appearing one by one, smooth transitions, app demo style"

Etapa 3: Criar Narração (Voice-over)

Use ElevenLabs ou Play.ht para gerar áudio profissional:

  1. 1. Copie o script de narração do roteiro
  2. 2. Escolha voz apropriada (profissional/casual/inspirador)
  3. 3. Ajuste configurações:
    • - Stability: 60-70
    • - Similarity: 80-90
    • - Style: 20-30
  4. 4. Gere e ouça preview
  5. 5. Faça ajustes se necessário (pausas, ênfase)
  6. 6. Baixe em WAV (melhor qualidade)

Etapa 4: Selecionar Música de Fundo

Fontes recomendadas:

Epidemic Sound

Biblioteca premium, ideal para comercial

Uppbeat

Grátis com atribuição

YouTube Audio Library

100% grátis, boa variedade

Suno AI

Gere música original com IA

Critérios: Tom condizente, sem vocal, boa para loop, não muito intrusiva

Etapa 5: Efeitos Sonoros (Opcional)

Adicione realismo e impacto:

  • • Whoosh (transições)
  • • Click/tap (interações de UI)
  • • Ambient (ambiente de fundo)
  • • Impact (momentos de destaque)

Fonte: Freesound.org, Zapsplat (grátis com atribuição)

⚡ Dicas de Produtividade

🎨 Batch Generation

Gere todas as imagens de uma vez no Midjourney/DALL-E antes de animar. Economiza tempo.

🔄 Reuse e Remix

Use mesma imagem com diferentes motion prompts para criar variedade.

📝 Anotações

Salve prompts que funcionaram bem. Construa biblioteca pessoal de prompts.

⏱️ Timing

Gere vídeos com 5-10s cada. Mais fácil de editar do que clips longos.
