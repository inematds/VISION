🎨 Aula 5.2 120 min

Manipulação avançada de imagens com IA

Domine técnicas profissionais de inpainting, outpainting, style transfer e manipulação criativa usando IA.

📖 Introdução

A manipulação avançada de imagens com IA revolucionou a edição fotográfica. Tarefas que antes levavam horas em Photoshop agora podem ser feitas em minutos com resultados surpreendentes. Nesta aula, você aprenderá técnicas que transformarão completamente seu workflow criativo.

Objetivo: Ao final desta aula, você será capaz de manipular imagens profissionalmente usando técnicas avançadas de IA como inpainting, outpainting e transferência de estilo.

🎯 Técnicas Fundamentais

🖌️ Inpainting (Preenchimento Inteligente)

Técnica para remover, substituir ou adicionar elementos dentro de uma imagem existente.

Casos de uso:

  • • Remover objetos indesejados (pessoas, cabos, marcas)
  • • Substituir elementos (trocar roupa, mudar objetos)
  • • Corrigir imperfeições (manchas, rugas, defeitos)
  • • Adicionar novos elementos que se integrem perfeitamente

🔲 Outpainting (Expansão de Canvas)

Técnica para expandir uma imagem além de suas bordas originais, criando conteúdo novo.

Casos de uso:

  • • Mudar proporção de imagem (quadrada para panorâmica)
  • • Adicionar mais contexto a uma cena
  • • Criar versões widescreen de imagens
  • • Expandir horizontes e paisagens

🎨 Style Transfer (Transferência de Estilo)

Aplicar o estilo artístico de uma imagem em outra, mantendo o conteúdo original.

Casos de uso:

  • • Transformar fotos em pinturas (Van Gogh, Monet, etc.)
  • • Aplicar estilos de anime ou cartoon
  • • Criar versões artísticas de produtos
  • • Unificar estética de múltiplas imagens

🔄 Image-to-Image (Transformação Guiada)

Usar uma imagem como base estrutural e modificá-la com prompts de texto.

Casos de uso:

  • • Transformar dia em noite (ou vice-versa)
  • • Mudar estações do ano
  • • Alterar clima e atmosfera
  • • Reimaginar cenários mantendo composição

🖌️ Tutorial: Inpainting com Adobe Firefly

Passo 1: Preparar Imagem no Photoshop

  1. 1. Abra sua imagem no Adobe Photoshop
  2. 2. Certifique-se de ter a versão mais recente do Photoshop (2024+)
  3. 3. Verifique se o Generative Fill está habilitado (beta features)

Passo 2: Selecionar Área

Use uma das ferramentas de seleção:

  • • Lasso Tool (L): Seleção livre à mão
  • • Object Selection Tool (W): Seleção automática de objetos
  • • Quick Selection Tool (W): Seleção por área
  • • Pen Tool (P): Seleção precisa (mais trabalhoso)
Dica: Faça uma seleção um pouco maior que o objeto para melhor blending

Passo 3: Generative Fill

  1. 1. Com a área selecionada, clique no botão "Generative Fill" que aparece
  2. 2. Digite um prompt descrevendo o que você quer:

Para remover:

Deixe o prompt vazio ou escreva "remove"

Para substituir:

"red vintage car" ou "wooden table"

Para adicionar:

"flying birds in the sky" ou "flowers on table"

3. Pressione "Generate" e aguarde (5-15 segundos)

4. O Firefly gerará 3 variações diferentes

Passo 4: Escolher e Refinar

  • • Navegue entre as 3 variações geradas
  • • Escolha a melhor ou clique "Generate" novamente para novas opções
  • • Cada geração cria uma nova camada (layer) automaticamente
  • • Ajuste opacidade ou use layer masks para blending manual
  • • Combine múltiplas gerações se necessário

Passo 5: Pós-processamento

Finalize com ajustes tradicionais:

  • • Color matching: ajuste cores para combinar com a imagem
  • • Blur edges: suavize bordas se necessário
  • • Dodge & Burn: ajuste luz e sombra
  • • Sharpen: aumente nitidez se necessário

📐 Tutorial: Outpainting com Leonardo.AI

Passo 1: Configurar Canvas

  1. 1. Acesse Leonardo.AI e vá para "Canvas Editor"
  2. 2. Faça upload da sua imagem
  3. 3. A imagem será posicionada no centro do canvas

Passo 2: Expandir Canvas

Defina a nova proporção:

  • Horizontal: Arraste as bordas laterais para fora
  • Vertical: Arraste bordas superior/inferior
  • Todos os lados: Para adicionar contexto ao redor
Proporções comuns: 16:9 (widescreen), 4:5 (Instagram), 1:1 (quadrado), 9:16 (vertical)

Passo 3: Prompts de Direção

Escreva um prompt descrevendo o que deve aparecer nas áreas expandidas:

Paisagem:

"continue the forest landscape, more trees and mountains in the distance, same lighting and atmosphere"

Interior:

"expand the room, show more furniture and walls, maintain modern minimalist style"

Retrato:

"show full body, same person, same outfit, studio background"

Passo 4: Ajustar Configurações

  • Image Strength: 0.3-0.5 (quanto menor, mais criativo; maior, mais fiel)
  • Prompt Strength: 6-8 (controle sobre o prompt)
  • Steps: 30-50 (qualidade vs velocidade)
  • Modelo: Escolha modelo apropriado (Leonardo Diffusion XL para realismo)

Passo 5: Gerar e Iterar

  1. 1. Clique em "Generate" e aguarde o resultado
  2. 2. Se não ficou perfeito, ajuste o prompt ou configurações
  3. 3. Você pode expandir em etapas (expandir um lado, depois outro)
  4. 4. Use inpainting para corrigir detalhes específicos
  5. 5. Download da imagem final em alta resolução

🎨 Style Transfer Profissional

Método 1: Midjourney Style Reference

Use o parâmetro --sref para aplicar estilo de uma imagem em outra:

/imagine [sua imagem URL] [seu prompt] --sref [imagem estilo URL] --sw 100

--sw controla a força do estilo (0-1000, padrão 100)

Método 2: Leonardo.AI Style Reference

  1. 1. No Leonardo, use "Image Guidance"
  2. 2. Upload sua imagem base (conteúdo)
  3. 3. Upload imagem de referência (estilo)
  4. 4. Ajuste "Style Strength" (quanto de estilo aplicar)
  5. 5. Prompt descrevendo o resultado desejado

Método 3: Stable Diffusion ControlNet

Para máximo controle usando ControlNet:

  • • Canny: preserva bordas e estrutura
  • • Depth: mantém profundidade da cena
  • • OpenPose: preserva poses de pessoas
  • • Combine múltiplos ControlNets para controle total

⚡ Dicas Profissionais

✅ Melhores Práticas

  • • Sempre trabalhe em alta resolução
  • • Salve versões intermediárias
  • • Seja específico nos prompts
  • • Use seleções precisas para inpainting
  • • Ajuste iluminação para combinar com original
  • • Faça múltiplas gerações e escolha a melhor
  • • Combine IA com edição manual para perfeição

❌ Evite

  • • Prompts vagos ou genéricos
  • • Ignorar consistência de luz e sombra
  • • Outpainting muito agressivo de uma vez
  • • Esquecer de ajustar perspectiva
  • • Usar primeira geração sem testar outras
  • • Não verificar resultado em 100% zoom
  • • Esquecer de combinar cores e tonalidades
